# scaffold/

The source of the **minimal skeleton** that `workbench-kit:generate-workbench` lays
into a user's repo. The engine (skills+utils+bin) comes from the installed `workbench`
plugin, so it is not here.

| Item | Role |
|---|---|
| `AGENTS.overlay.md` | persona defaults — regenerated by interview, edited by the user |
| `templates/` | harvest entry / task-AGENTS formats (profile, user-editable) |
| `.github/` | required issue elements, PR skeleton (profile) |
| `docs/` | empty knowledge skeleton (index + empty anchor tables + log) — empty-start |
| `codebases.yaml` | empty target manifest |

On generation, `AGENTS.md` = the plugin's `AGENTS.core.md` composed with this
`AGENTS.overlay.md`. (The composer is provided by generate-workbench — to be built.)

Operational prose produced into the user repo follows the persona `language`
(`AGENTS.overlay.md`); this framework scaffold itself is in English.
